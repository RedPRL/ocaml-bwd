Module Bwd.Bwd

This module is similar to List but for backward lists.

Notes on the discrepancies with List:

  • New:

    • prepend was added as the forward version of append.
    • to_list and of_list were added for conversions between standard lists.
    • Infix was added for the infix notation.

  • Changed:

    • cons was replaced by snoc.
    • append was replaced by a new version that performs the "textual order yoga".
    • All indices count from the right rather than the left: nth, nth_opt, init, iteri, mapi, and filteri.
    • All iteration functions work from the right rather than the left: iter, map, fold_left, fold_right, exists, mem, find, filter, and other similar functions.

  • Forbidden:

    • Functions rev, rev_append, rev_map, and rev_map2 will never be included.

  • Missing but may be added in the future:

    • tl (as hd for lists), hd (as tl for lists), concat, flatten, concat_map.
    • Functions related to association lists.
    • Functions related to sorting.
    • Conversion from/to sequences.

Please open a GitHub issue if you want a function to be included. We want to make this library useful to you, too!
